# TranscripTonic Documentation

Welcome to the TranscripTonic documentation. Here you will find guides on how to install, use, and integrate the extension.

## Guides
- [**Installation Guide**](INSTALLATION.md): How to install the extension in Developer Mode.
- [**Usage Guide**](USAGE.md): How to view, manage, and download transcripts.
- [**Integration Guide**](INTEGRATION.md): How to send transcripts to your external React/Node.js application using webhooks.

## Project Structure
- `extension/`: The core Chrome extension source code.
- `assets/`: Images and screenshots used in documentation.
- `types/`: TypeScript type definitions for the extension.
- `docs/`: Comprehensive guides (you are here).
